On-premise deployment
Docker Compose stack on an idle GPU (A2) at 192.168.50.85 — no new cloud spend, no contention with production workloads.
Isolated & safe
One-command deploy
Same codebase runs on a laptop (SQLite + fallback models) or the server (Postgres + Chronos-2 GPU). Single deploy.sh gets you live.
Env-switched
Security model
PBKDF2-hashed passwords, short-lived session cookies, pending-approval signups, guarded actions requiring confirmation for all writes. TLS-ready for LAN-external access.
Enterprise-grade
Role-based access control
RoleCapabilities
AdminFull system access, user management, settings
ManagerView all + approve reorders, run pipeline
AnalystView all data, create reports, query
ViewerRead-only dashboard and report access
